# Arduino Nano Line-Following Robot Project

## 구현된 기능
- ESP32로부터 JSON 형식 UART 수신 (DEST, INSERT_OBS, REMOVE_OBS 명령어)
- 모터 제어 및 라인트레이싱 기본 동작
- 센서 제어 (라인 센서 4개)
- 디버깅용 부저 (100ms)
- BFS 알고리즘을 이용한 최단 경로 계산
- 회전 최소화 경로 선택 (직진, 좌회전, 우회전, U턴 순서)
- 초음파 센서를 이용한 전방 장애물 감지 (10cm 이내)
- 장애물 감지 시 일시정지 기능
- 명령어 처리 함수 분리 (리팩토링 완료)
